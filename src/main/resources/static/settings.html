缓存管理

清理缓存不会删除原始文档和知识库记录，但可能导致下次处理变慢。缓存可以重新生成。

垃圾箱与数据生命周期

  • ACTIVE → TRASHED → PURGED：删除后进入垃圾箱，7 天内可恢复。
  • 永久删除会清理原始文本、片段、向量与相关缓存。
  • 前往 垃圾箱 管理已删除文档。

日志与说明
