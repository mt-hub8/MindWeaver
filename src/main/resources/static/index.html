5 分钟体验路径

  1. 启动服务（见 README「本地快速启动」）。
  2. 打开 文档管理，上传 .txt / .md / 文本型 .pdf。
  3. 等待摄入任务状态变为「已完成」（后台自动切块、生成向量、写入索引）。
  4. 打开 知识库问答 提问，查看检索引用（Citation）。
  5. （可选）按 检索质量评估 说明运行检索评估。

系统能力说明

  • RAG 问答与检索引用（Citation） 回答附带引用片段，帮助用户追踪来源、降低幻觉风险。
  • 异步文档摄入流水线 上传后快速返回任务 ID，后台完成切块、向量生成与索引写入。
  • Embedding 缓存 / 去重 相同文档片段不重复计算向量，节省成本与延迟。
  • 检索质量评估（Evaluation） 用 HitRate、Recall、Precision、MRR 等指标判断检索质量。
  • 两阶段检索与重排（Rerank） 先扩大候选集再重排，提升关键词相关片段的最终排名。
  • 混合检索融合（Hybrid） 兼顾语义相似（dense）与关键词匹配（lexical），经 RRF 融合。
  • 统一 API 错误响应 统一 code / message / traceId，便于前端与运维排查。
