知识库问答

基于已摄入文档提问，系统返回带检索引用（Citation）的回答，帮助核对来源、降低幻觉风险。

知识库生命周期过滤说明

  • 知识库问答只会使用「已启用」的文档；已删除文档不会再进入回答引用。
  • 重新建立索引后，系统只会使用最新有效的文档片段（当前索引版本）。
  • 旧版本片段（已被新索引替代）不会再进入回答引用，即使底层向量尚未物理删除。
  • 如需管理文档状态，请前往 文档管理 进行删除或重新建立索引。

上传文档后提问（Upload-to-Ask）

  1. 前往 文档管理 页面上传文档。
  2. 等待处理状态变为「已完成」（系统会异步完成切块、向量生成与知识库索引写入）。
  3. 回到本页「知识库问答」，输入与文档相关的问题。
  4. 查看答案与检索引用（Citation），核对检索 metadata。
