Upload-to-Ask 流程

  1. 打开 Documents 页面。
  2. 上传 .txt、.md 或文本型 .pdf（不支持扫描版 OCR）。
  3. 等待上传成功 summary（chunk / embedding / vector 写入完成）。
  4. 返回本页 Ask，输入与文档相关的问题。
  5. 查看 Answer 与 Citations，核对 retrieval metadata。
