常见问题

为什么问答没有结果？

请确认已上传文档且处理完成，并检查所选知识库分组内是否有可用文档。

为什么模型调用失败？

请在模型设置中测试连接，确认 API Key 或本地 Ollama 服务正常。

为什么文档处理失败？

检查文件格式与大小限制，可在文档管理查看处理记录与错误详情。

如何提升检索质量？

优化文档切分、选择合适分组、调整 TopK，并参考质量诊断页建议。

术语速览

RAG
检索增强生成：先检索知识库再生成回答。
Embedding
向量表示：将文本转为可相似度检索的向量。
TopK
召回数量：检索时取相似度最高的 K 个片段。
Recall@K
召回率：相关文档是否出现在前 K 个结果中。
Context Precision
上下文精准度：送入模型的片段是否真正相关。
